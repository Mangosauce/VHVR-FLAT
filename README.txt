VHVR FlatVR Alpha 0.1.0

Experimental modified ValheimVR DLL.

Install:
1. Install BepInEx for Valheim.
2. Install ValheimVR 0.9.2100.
3. Go to BepInEx/plugins/ValheimVR/.
4. Back up the original ValheimVRMod.dll.
5. Replace it with this ValheimVRMod.dll.
6. Launch through SteamVR.

Controls:
F8 = Toggle FlatVR / VR controller mode

Known issues:

1.  Main menu UI may become zoomed/corrupted after returning to main menu from game. May appear zoomed in or cursor may not function properly. Recommended restart of game or use keyboard keys to navigate into game.
2. Giant helmet rendering bug. You'll know it when you see it. If It happens, de-equip and re-equip helmet.
3. HUD hide controls do not fully work in FlatVR.
4. Mouse Y-axis aiming is not implemented.
5. Some movement/orientation state may need manual VHVR recentering after mode switches with the Home hotkey.
